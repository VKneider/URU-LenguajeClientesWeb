Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo que define la presentación de documentos HTML. Permite separar el contenido de la presentación, lo que facilita el mantenimiento y la coherencia visual de un sitio web.

En esta guía, exploraremos los conceptos fundamentales de CSS que todo desarrollador web debe conocer para crear interfaces atractivas y funcionales.

Nota: Para utilizar CSS, puedes incluirlo en tu HTML de tres formas: en línea con el atributo style, en la sección <head> utilizando la etiqueta <style>, o en un archivo externo vinculado con <link>.

Selectores CSS

Los selectores son patrones que se utilizan para seleccionar los elementos HTML que deseas estilizar. Hay varios tipos de selectores que permiten apuntar a elementos específicos dentro de tu documento.

Tipos principales de selectores

/* Selector de elemento */
p {
    color: blue;
}

/* Selector de clase */
.destacado {
    font-weight: bold;
}

/* Selector de ID */
#cabecera {
    background-color: gray;
}

/* Selector de atributo */
input[type="text"] {
    border: 1px solid #ccc;
}

/* Selector descendiente */
article p {
    font-size: 16px;
}

/* Selector hijo directo */
ul > li {
    list-style: square;
}

/* Selector de hermano adyacente */
h2 + p {
    font-style: italic;
}

/* Selector universal */
* {
    box-sizing: border-box;
}

Combinando selectores

Los selectores se pueden combinar para crear reglas más específicas y potentes:

/* Elementos que tienen ambas clases */
.clase1.clase2 {
    color: purple;
}

/* Elementos con una clase específica dentro de un ID */
#contenedor .item {
    margin: 10px;
}

/* Elemento específico con una clase */
p.nota {
    font-style: italic;
}

Consejo: Utiliza selectores específicos cuando necesites dirigirte a elementos concretos, pero evita ser demasiado específico innecesariamente, ya que dificultará la sobreescritura de estilos.

Especificidad en CSS

La especificidad es un conjunto de reglas que determina qué estilos se aplican a un elemento cuando hay múltiples reglas que apuntan al mismo elemento. Es crucial entender la especificidad para evitar confusiones cuando los estilos no se aplican como esperamos.

Orden de especificidad (de menor a mayor)

Nivel Tipo de selector Valor Ejemplo
1 Selectores de elemento y pseudo-elementos 0,0,0,1 p, h1, ::before
2 Selectores de clase, atributos y pseudo-clases 0,0,1,0 .clase, [type="text"], :hover
3 Selectores de ID 0,1,0,0 #id
4 Estilos en línea 1,0,0,0 style="color: red;"
5 !important ∞ color: red !important;

Ejemplo de especificidad

/* Especificidad: 0,0,0,1 */
p {
    color: black;
}

/* Especificidad: 0,0,1,0 */
.clase-media {
    color: blue;
}

/* Especificidad: 0,1,0,0 */
#id-alta {
    color: red;
}

Advertencia: El uso de !important rompe el flujo natural de especificidad y puede conducir a código CSS difícil de mantener. Úsalo con moderación y sólo cuando sea absolutamente necesario.

Box Model (Modelo de Caja)

El modelo de caja es un concepto fundamental en CSS que describe cómo se representan visualmente todos los elementos HTML. Cada elemento se considera como una caja rectangular compuesta por cuatro áreas principales:

Contenido

Componentes del Box Model

  1. Content (Contenido): El área donde se muestra el texto, imágenes u otros contenidos.
  2. Padding (Relleno): El espacio transparente entre el contenido y el borde.
  3. Border (Borde): Una línea que rodea el padding y el contenido.
  4. Margin (Margen): El espacio transparente que separa un elemento de otros elementos.

Propiedades del Box Model

.elemento {
    /* Dimensiones de contenido */
    width: 300px;
    height: 200px;
    
    /* Padding (arriba, derecha, abajo, izquierda) */
    padding: 20px;
    /* O individualmente */
    padding-top: 10px;
    padding-right: 15px;
    padding-bottom: 10px;
    padding-left: 15px;
    
    /* Border */
    border: 2px solid black;
    /* O individualmente */
    border-width: 2px;
    border-style: solid;
    border-color: black;
    
    /* Margin */
    margin: 15px;
    /* O individualmente */
    margin-top: 10px;
    margin-right: 20px;
    margin-bottom: 10px;
    margin-left: 20px;
}

La fórmula para calcular el ancho total de un elemento es:

ancho total = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right

La fórmula para calcular el alto total de un elemento es:

alto total = height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom

Box-sizing

La propiedad box-sizing cambia cómo se calcula el ancho y alto de un elemento:

/* El valor predeterminado */
.content-box {
    box-sizing: content-box;
    /* width y height solo incluyen el contenido */
}

/* Incluye padding y border en width y height */
.border-box {
    box-sizing: border-box;
    /* width y height incluyen contenido, padding y border */
}

Consejo: Muchos desarrolladores aplican box-sizing: border-box; a todos los elementos para un cálculo de dimensiones más intuitivo:

* {
    box-sizing: border-box;
}

Clases e IDs

Las clases y los IDs son atributos HTML que permiten identificar y estilizar elementos específicos. Son fundamentales para aplicar estilos de manera selectiva en tu página web.

Clases

Las clases son atributos que pueden aplicarse a múltiples elementos. Se definen con un punto . en CSS.

/* En tu CSS */
.boton {
    padding: 10px 15px;
    background-color: #3b82f6;
    color: white;
    border-radius: 4px;
}

.boton-grande {
    font-size: 18px;
    padding: 15px 20px;
}

/* Puedes aplicar múltiples clases a un elemento */

En tu HTML:

<button class="boton">Botón Normal</button>
<button class="boton boton-grande">Botón Grande</button>

Resultado:

Ventajas de las clases:

  • Pueden reutilizarse en múltiples elementos
  • Un elemento puede tener múltiples clases
  • Son ideales para estilos que se comparten entre varios elementos

IDs

Los IDs son identificadores únicos que solo deben aplicarse a un elemento por página. Se definen con un hash # en CSS.

/* En tu CSS */
#encabezado-principal {
    font-size: 32px;
    color: #1e40af;
    text-align: center;
}

#formulario-contacto {
    background-color: #f3f4f6;
    padding: 20px;
    border-radius: 8px;
}

En tu HTML:

<h1 id="encabezado-principal">Título Principal</h1>
<form id="formulario-contacto">...</form>

Los IDs tienen mayor especificidad que las clases, por lo que sus estilos prevalecerán sobre los estilos de clases que se apliquen al mismo elemento.

Importante: Aunque técnicamente un navegador mostrará una página con IDs duplicados, esto es incorrecto según los estándares HTML. Cada ID debe ser único en la página, especialmente si utilizas JavaScript para manipular elementos por ID.

¿Cuándo usar clases vs IDs?

  • Usa clases para estilos que se repiten o que pueden aplicarse a múltiples elementos.
  • Usa IDs para elementos únicos en la página (como el encabezado principal) o elementos que necesitas manipular individualmente con JavaScript.

Propiedad Display

La propiedad display es una de las más importantes en CSS, ya que determina cómo se muestra un elemento en el flujo del documento. Controla el comportamiento de renderizado del elemento y cómo interactúa con otros elementos.

Valores comunes de Display

display: block

.display-block {
    display: block;
    background-color: #dbeafe;
    padding: 10px;
    width: 100%;
}

Características:

  • Ocupa todo el ancho disponible de su contenedor
  • Comienza en una nueva línea
  • Respeta propiedades de ancho y alto

Ejemplo:

Elemento block
Otro elemento block

display: inline

.display-inline {
    display: inline;
    background-color: #e0e7ff;
    padding: 10px;
}

Características:

  • Ocupa solo el espacio necesario para su contenido
  • No comienza en una nueva línea
  • No respeta propiedades de ancho y alto
  • El padding horizontal funciona, pero el vertical puede superponerse

Ejemplo:

Elemento inline Otro elemento inline

display: inline-block

.display-inline-block {
    display: inline-block;
    background-color: #fae8ff;
    padding: 10px;
    width: 150px;
    height: 50px;
}

Características:

  • Combinación de block e inline
  • No comienza en una nueva línea
  • Respeta propiedades de ancho y alto
  • Respeta padding, margin y border completamente

Ejemplo:

Inline-block
Otro inline-block

display: none

.display-none {
    display: none;
}

Características:

  • El elemento no se muestra en absoluto
  • No ocupa espacio en el diseño
  • Útil para elementos que quieres ocultar completamente

Conclusiones y Recursos Adicionales

Esta guía introduce los conceptos fundamentales de CSS que forman la base para crear diseños web atractivos y funcionales. Dominar estos conceptos te permitirá construir interfaces más complejas y responsivas.

Conceptos clave a recordar

  • Los selectores te permiten apuntar a elementos específicos
  • La especificidad determina qué reglas CSS prevalecen cuando hay conflictos
  • El Box Model describe la composición de cada elemento HTML
  • Las clases e IDs te permiten aplicar estilos de manera selectiva
  • La propiedad display controla cómo se comportan los elementos en el flujo del documento

Pasos siguientes

  • Flexbox y Grid en profundidad
  • Posicionamiento CSS
  • Transiciones y animaciones
  • Media queries y diseño responsivo
  • Variables CSS (Custom Properties)
  • Preprocesadores como Sass o Less

Recuerda: La práctica constante es la clave para dominar CSS. Intenta recrear diseños que encuentres interesantes y experimenta con las propiedades que has aprendido.
