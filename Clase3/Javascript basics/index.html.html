Fundamentos de JavaScript

1. Tipos de variables en JavaScript

JavaScript tiene tres formas principales de declarar variables:

  • var: Forma antigua, tiene ámbito de función y puede causar problemas de hoisting.
  • let: Permite declarar variables con ámbito de bloque. Es la forma recomendada para variables que cambian.
  • const: Declara constantes (valores que no cambian).

Ejemplo:


    var nombre = "Juan"; // Variable global o de función
    let edad = 25;       // Variable con ámbito de bloque
    const pais = "México"; // Constante, no puede ser reasignada
    console.log(nombre, edad, pais);

2. Hoisting

El hoisting es el comportamiento de JavaScript en el que las declaraciones de variables y funciones son "movidas" al inicio del código.

Ejemplo:


    console.log(x); // undefined (por hoisting, la declaración se mueve arriba, pero no su asignación)
    var x = 10;
    console.log(x); // 10

Con let y const, el hoisting ocurre, pero no se pueden usar antes de declararlas:


    console.log(y); // Error: Cannot access 'y' before initialization
    let y = 20;

3. Tipos de datos en JavaScript

JavaScript tiene varios tipos de datos primitivos:

  • string: Cadena de texto. Ejemplo: "Hola"
  • number: Número. Ejemplo: 42
  • boolean: Verdadero o falso. Ejemplo: true, false
  • undefined: Variable declarada pero sin valor.
  • null: Valor nulo intencional.
  • symbol: Tipo único introducido en ES6.
  • object: Colección de pares clave-valor (ver siguiente sección).

Ejemplo:


    let texto = "Hola, mundo"; // string
    let numero = 42;           // number
    let verdadero = true;      // boolean
    let sinValor;              // undefined
    let nulo = null;           // null
    console.log(typeof texto, typeof numero, typeof verdadero, typeof sinValor, typeof nulo);

4. Objetos en JavaScript

Un objeto es una estructura que permite almacenar datos en pares clave-valor. Puede contener propiedades (datos) y métodos (funciones).

Ejemplo:


    const persona = {
        nombre: "Ana",
        edad: 30,
        saludar: function() {
            console.log("Hola, soy " + this.nombre);
        }
    };

    console.log(persona.nombre); // "Ana"
    persona.saludar();           // "Hola, soy Ana"
    persona.apellido = "Pérez";  // Agregar propiedad
    delete persona.edad;         // Eliminar propiedad
    console.log(persona);

5. JSON (JavaScript Object Notation)

JSON es un formato ligero basado en texto para intercambiar datos. Es muy utilizado para transferir información entre cliente y servidor.

Ejemplo:


    // Objeto en JavaScript
    const objeto = { nombre: "Luis", edad: 28 };

    // Convertir un objeto a JSON
    const jsonString = JSON.stringify(objeto);
    console.log(jsonString); // '{"nombre":"Luis","edad":28}'

    // Convertir JSON a un objeto
    const nuevoObjeto = JSON.parse(jsonString);
    console.log(nuevoObjeto.nombre); // "Luis"
